# qa-equivalence-testing

## 🧪 Equivalence Class Testing

## 📌 Descripción
Proyecto de testing manual enfocado en la aplicación de la técnica **Clases de Equivalencia** para validar campos de entrada en una aplicación web.

Este proyecto fue desarrollado como parte de mi formación en QA Engineer.

---

## 🎯 Objetivo
Validar el comportamiento del sistema mediante la identificación de datos válidos e inválidos utilizando técnicas de diseño de pruebas.

---

## ✅ Actividades realizadas
- Identificación de clases válidas e inválidas
- Definición de escenarios de prueba
- Validación de reglas de negocio
- Documentación de resultados

---

## 🛠️ Herramientas
- Manual Testing
- Google Sheets / Excel

---

## 📚 Técnicas QA aplicadas
- Equivalence Partitioning
- Input Validation Testing
